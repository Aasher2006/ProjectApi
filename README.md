# Airline Management System

A full-stack airline management application developed using HTML, CSS,
JavaScript, Node.js, Express.js and PostgreSQL.

## Features

- Dashboard analytics
- Flight management
- Passenger management
- Booking management
- Aircraft management
- Staff management
- Maintenance records
- Payment tracking
- REST API integration
- PostgreSQL database

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
